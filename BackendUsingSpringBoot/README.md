# First-Springboot-project

This project is a part of my Spring Boot learning journey.
It is a full stack application built using:

* Backend: Spring Boot
* Frontend: React
* Database: MongoDB (planned/current)
* Build Tool: Maven

##  Project Overview

The application is a Journal Management System where users can create, view, update, and delete journal entries. The main purpose of this project is to learn and implement core concepts of Spring Boot and full stack development.

##  Features

* Create journal entries
* View all journal entries
* Update existing entries
* Delete journal entries
* REST API integration
* Frontend and backend communication

## Technologies Used

### Backend

* Java
* Spring Boot
* Spring Web
* Spring Data MongoDB / JPA
* Maven
